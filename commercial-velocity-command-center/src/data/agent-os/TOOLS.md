# TOOLS

## sessions_spawn
Delegate work to departments. Include:
- Department name
- Files to read
- Exact deliverables

## memory_search / read / write / edit
Manage knowledge files.

## cron
Schedule recurring tasks.

## SUB-AGENT LIMITS
Sub-agents cannot:
- Spawn other sub-agents
- Schedule cron
- Access gateway/nodes

Sub-agents can:
- Read/write files
- Search web
- Run commands
