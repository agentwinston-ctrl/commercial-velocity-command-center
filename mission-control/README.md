# Mission Control (local)

A tiny local dashboard (no dependencies) that shows progress from `mission-control/tasks.json`.

## Run

```bash
cd /Users/winston1/.openclaw/workspace/mission-control
node server.js
```

Then open:
- http://127.0.0.1:3033

## Edit tasks

Edit `mission-control/tasks.json` and refresh the page.

## API

- http://127.0.0.1:3033/api/tasks
