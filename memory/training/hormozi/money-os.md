# Money Model OS (100M Money Models) — CRE

## Customer Financed Acquisition (CFA)
- Paid trial covers ad spend (no out-of-pocket).
- Setup fee covers CAC + profit.

## Pricing Model (default)
- $1,500 paid trial (ad spend)
- $8,000 setup
- $500 per booked estimate

## Core KPI Tree
- CAC (to paid trial)
- Conversion: trial → setup
- Appointment economics: cost per booked estimate vs expected gross profit per job
